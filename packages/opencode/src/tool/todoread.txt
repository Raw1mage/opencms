Read the current to-do list for the session.

## Parameters

This tool takes no parameters. Leave the input empty (`{}`); do not include a dummy key like `"input"` or `"empty"`.

## Returns

A list of todo items with their `id`, `content`, `status`, and `priority`. Returns an empty list if no todos exist yet.

## Frequency / trigger rules

This is a tool schema. Rules about WHEN to call this tool live in the system prompt and agent driver prompts, not here.
