# Project `.opencode` Legacy Examples

This directory stores migrated examples that previously lived under repo root `.opencode/`.

## Purpose

- Preserve historical command/agent/tool/theme examples as template assets.
- Avoid keeping non-runtime legacy payload in repo root `.opencode/`.

## Important

- These files are **examples/reference assets**, not auto-loaded runtime defaults.
- Runtime project instructions use `<project>/AGENTS.md`.
